# Create React App Ts

## Start with local

### Prerequisite

Install following dependencies beforehand.

- Setup git
- Setting vscode
- Install nodejs

### Start project

```bash
git clone ${REPO_URL}
cd /path/to/project
npm install
npx mrm lint-stage

npm run start
```

Go [http://localhost:3000](http://localhost:3000) to open web.
